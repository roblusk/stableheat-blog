The Bowa project is a clean reference example of a residential roof deicing installation. We documented it from the property exterior down to the element placement, transformer configuration, and the finished system ready for its first winter.

This is the kind of project we use as a template when prospective customers ask "what does a typical residential roof deicing install actually look like?" The answer is something close to Bowa.

## The property

The home itself is a substantial residential structure: white-brick exterior, slate roof, multi-section roof geometry with the kind of dormer-and-gable details that look beautiful in summer and create ice-dam vulnerability in winter. The pool deck visible in the rear of the property is one of those details. The roof above it has natural snow accumulation patterns that, untreated, would feed water issues right where the homeowner least wants them.

A property like this has too much architectural value to risk to ice damage. That's the brief.

## What the install covers

The Bowa system follows the standard residential roof deicing pattern, scaled to the property:

- **Eave runs** along the perimeter where ice dams typically form, with cable spacing tuned to the snow-load conditions of the local climate
- **Valley details** where multiple roof planes meet, the high-collection zones where untreated systems fail first
- **Gutter integration** to keep the meltwater path open from eave to downspout to grade
- **Transformer and control infrastructure** sized for the full system load, with environmental sensors triggering operation only when needed

![Wide view of the conservatory eave with the protective membrane fully bonded over the heating element along the lower edge of the slate roof.](./images/conservatory-roof-detail.jpg)

*The conservatory eave with the membrane bonded over the cable run. Once the trim and copper gutter go back on, none of this is visible from the ground.*

The element itself is StableHeat self-regulating cable. The cable adjusts current draw locally based on conditions, pulling more under snow accumulation, backing off in clear weather, never overheating itself even where it's wrapped tight against fittings.

![Two toroidal step-down transformers mounted on an aluminum panel next to a blue weather-controller, with cabling routed through a basement utility space.](./images/transformer-controller.jpg)

*The system brain in the basement: dual transformers stepping power down to the safe low-voltage cable, with the weather controller deciding when the system runs.*

> A residential roof deicing system isn't an emergency device. It's an automatic one, and the automatic part is what makes it work.

For homeowners evaluating roof deicing on a similar property, the value of walking the install end to end is concrete. You see what gets installed, where, and what the resulting system looks like at handover. It closes the gap between "we install roof deicing" and "here's what it actually looks like on your specific roof type."

The Bowa project represents the residential standard we now apply across most multi-section homes. Each project varies in detail (roof pitch, climate severity, finish material), but the underlying installation pattern is consistent.

For homeowners with substantial residential properties in ice-dam-prone regions: this is the install pattern to ask about. Not "do you do roof deicing" but "what does a Bowa-style install look like on my roof."
